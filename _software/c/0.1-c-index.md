# 为什么要学习 C

* 出色的可移植性，可以在多种平台上运行。实际上，是否支持 C 已经成为检验一个平台是否成熟的一个标准。
* 运行效率很高，可以直接和操作系统、硬件打交道
* 相比起其他底层语言更加简洁
